Upload a Song for Genre Prediction

Model Information:

This model predicts the genre of a song using a Convolutional Neural Network (CNN) with a 82.59% accuracy.

Trained on the GTZAN dataset, the model can classify songs into 10 genres: metal, classical, pop, country, disco, reggae, blues, jazz, rock, and hiphop.

For more information on how the model was trained, visit the GitHub project page.

Supported File Types: Only 'wav', 'mp3', 'flac', and 'aac' files are supported. Max file size : 30 Mb
